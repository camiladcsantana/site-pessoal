Dicionário HTML

Um dicionário básico com algumas das principais tags HTML e seus significados:

Estrutura básica:

<html>: Define o documento como um arquivo HTML.
<head>: Contém metadados sobre o documento, como título e links para scripts e folhas de estilo.
<title>: Especifica o título do documento, mostrado na aba do navegador.
<body>: Contém o conteúdo visível da página.

Cabeçalhos e Textos:

<h1> a <h6>: Representam títulos e subtítulos, onde <h1> é o mais importante e <h6> o menos.
<p>: Define um parágrafo.
<br>: Insere uma quebra de linha simples.
<b>: Aplica negrito ao texto.
<i>: Aplica itálico ao texto.
<u>: Sublinha o texto.
<b>:Indica texto de forte importância, geralmente exibido em negrito.

Links e Imagens:

<a>: Cria um link de hipertexto.
<img>: Insere uma imagem.
<map> e <area>: Usados para criar mapas de imagem clicáveis.

Semântica e Estrutura:

<header>: Define um cabeçalho para um documento ou seção.
<nav>: Define um conjunto de links de navegação.
<section>: Define uma seção no documento.
<aside>: Define conteúdo lateral.
<footer>: Define um rodapé para um documento ou seção.
